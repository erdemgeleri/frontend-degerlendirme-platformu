1)Uygun npm sürümünü indirmek için aşağıdaki komutu girin
	```bash
	npm install -g npm@10.8.1
2)Proje bağımlılıklarını indirmek için aşağıdaki komutu vscode terminaline girin.
	npm install
3)Yukarıdaki komut çalıştırıldıktan sonra projeye uyumlu node sürümünü indirmek için vscode terminaline aşağıdaki komutu girin.
	nvm install 20.16.0
4)İlgili node sürümünü kullanmak için aşağıdaki komutu terminale girin.
	nvm use 20.16.0
5)Yükleme işlemlerini tamamladıktan sonra projenin içine girmek için aşağıdaki komutu vscode terminaline girin.
	cd viteapp
6)Geliştirme sunucusunu başlatmak ve projeyi yerel olarak çalıştırmak için vscode terminaline aşağıdaki komutu girin
	npm run dev
